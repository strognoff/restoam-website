# RestoAM Marketing Website

Marketing website for RestoAM - Restaurant Asset Management platform.

## Tech Stack

- **React 19** - UI framework
- **Vite 7** - Build tool
- **Tailwind CSS 4** - Styling
- **React Router 7** - Navigation

## Pages

| Route | Description |
|-------|-------------|
| `/` | Landing page with hero, features, testimonials, CTAs |
| `/pricing` | Pricing plans and FAQ |
| `/faq` | Frequently asked questions |
| `/privacy` | Privacy Policy |

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deployment

Configured for Vercel/Netlify deployment.
